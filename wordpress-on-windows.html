My LAMP Stack

The primary tool in my dev is Desktopserver from ServerPress. This is what works for me running on my Win10 laptop with 4 GB RAM. You can get up and running for free to work on your one or two sites Premium adds some new tool automations and support for unlimited development websites.

Editor Preference

  • Sublime Text 3 - Eval Free / Buy $70

FTP Client - Really?

  • Yes - I insist!
  • FileZilla - Just get it
  • No editing files using the dashboard editor
You need to learn to use this tool. There are lots of options for FTP clients. Commodity tool. Open source is good. Repeat after me - I will not edit files through the WordPress dashboard. I will edit them in my code editor and then save them to my website.

DesktopServer

  • "Buy" Free Version
  • Download & Install
  • Windows Installs to c:\xampplite

Setup DesktopServer

Not going to go thorugh the DTSVR install. Its a typical Windows install. When running, you need to Run As Administrator. End up with a c:\xampplite folder. Not changeable. Find your WAMP stack in here. Apache. MySQL. PHP 5.5.24

Build First Site - WordPress 5 Minute Install

  • Start DesktopServer
  • Create Shortcut => c:\xampplite\DesktopServer.exe
  • Run as Administrator
  • Port 80 Conflict => Skype?
Suggest you create a destkop shortcut to the executable, so reduce the need to search. In shortcut properties, select "Run as administrator". Occasionally you will see a White Screen on the DTSVR display. It uses port 80 for the local lan. Latest versions of Skype grab this port for inbound comm. Skype can be configured to use an alternative port.
Enter name of my new site and use default location or browse to a new one.
Awesome plugin included with DTSVR. Bypass login so I don't have to remember my Dev passwords or even bother storing them in my password manager.

Code Editing / Development

  • Open Local Project Folder in Sublime Text
  • Save Sublime Project
  • Edit PHP and CSS as needed - Save
  • Create files as needed - Save
  • PHP aware - syntax highlighting
Pulling in the entire folder tree offers availability to access all files, even if you have no need to edit them. I find it very helpful when searching for WordPress hooks using the "find in open folders" search.
Edit any of the php, css or JS files in your project. create new directories and files. Build your entire customization layout right from the editor side panel.
