Things to Look For

—Backup Plugins—

  • Automatic Backups
  • Multiple Destinations
  • Various Backup Types - full, db, media
  • Restoration
  • Email Notifications on Failure
Set it once and forget about it. Don't save your backup on your web host. If your host goes down, there goes your backup. Set up a free DropBox account. Amazon S3 is VERY inexpensive. Take db backups frequently, full backups less, media only on occasion - depending on site activity. Need to be able to easily restore what has been backed up. Get notified when things are going south.
