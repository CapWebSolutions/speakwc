Common WordPress Problems

  • White Screen of Death
  • Error Establishing a Database Connection
  • Maintenance Mode Forever
  • Internal server error
  • Syntax Error in WordPress
Here are a few of the most common issues with WP sites. How many of you have experienced these? We are going to talk about how to troubleshoot a few of these and get into some Q&A on others.

White Screen of Death

  • Memory Issues
  • Plugin / Theme Conflicts

Error Establishing a Database Connection

  • Error talking to WP db
  • Login credentials
  • Db corrupt
  • MySQL issue on host
Check the wp-config.php file - verify db config is correct

Syntax Error in WordPress

  • Pasted Code Snippet
  • Need FTP to fix
  • Remove repalce new code
Probably the result of a incorrectly pasted code snippet. Extra bracket, missing ;
