I wanted to start with a back to basics approach to web design. Looking at a foundation and building up from there. I build web sites using WordPress and the Genesis framework. The framework enables me to work within a comfortable set of features so that basic dev is quick and flexible. From there I build up with the customizations needed. I thought that building on a CSS framework would be a benefit - since CSS is my challenge.

Intro to Bootstrap

  • what it is
  • why use it
  • what can you do with it
  • where to get it
  • how to use it
For the first few sessions, I would like us to talk about the Bootstrap framework. Learn a little about what it is and how to use it. Things we can do with it. To kick it off let's take a look at the what, where and how ...
Here's the splash screen for the get bootstrap website. Bootstrat is an open source project so there's no licensing costs to use it in your personal projects or commercail projects. It's described as a framework for HTML, CSS and javascript. I think the most important part of this is that it focuses on responsive web design.

Get it & Use It

  • Link to CDN or Content delivery network

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap
/3.3.6/css/bootstrap.min.css">
  • Download and Store Locally - Your Own Copy
    • Bootstrap files in your web server or local computer
      • 
<link rel="stylesheet" href="css/reveal.css">
    • Put bootstrap into same directory tree as your index.html

Get Started

  • Basic HTML - index.html

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1">
<title>Jumbotron Template for Bootstrap</title>
</head>
<body>
<h1 class="display-class-3">Hello, WebDes SIG!</h1>
</body>
</html>
