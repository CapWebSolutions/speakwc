Goal For Session

  • Create environment on a Windows 10 box that:
  • Familiar Windows UI
  • Robust toolset
  • Duplicable
  • Low cost
  • Includes features such as:
    1. Local build & testing
    2. Smart file editing
    3. FTP connection to host server
    4. Deployment

This is What I Have in Mind

  • Local by Flywheel
  • Visual Studio Code
  • FileZilla FTP Client
  • Git Version Control
  • WP Migrate DB / Pro
To put it more specifically, the workflow include local storage structure, full site local testing with smart editing tools, version control, easy file + db upload and finally deployment/upload to public interwebs with minimal headaches
