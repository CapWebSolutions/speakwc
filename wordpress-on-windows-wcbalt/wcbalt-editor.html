  • Intellisense - Code aware - syntax highlighting, auto-complete
VSC runs on Windows, Linux, and even macOS. offer link to Download LBF - Right Click show Folder - Drill to Public folder VSC has nice set of extensions for PHP and WordPress New versions each month

VSC - Open website project folder

Pulling in the entire folder tree offers availability to access all files, even if you have no need to edit them. I find it very helpful when searching for WordPress hooks using the "find in open folders" search.

VSC - Browse to theme folder

Edit any of the php, css or JS files in your project. create new directories and files. Build your entire customization layout right from the editor side panel. Editing functison.php to add that perfect snippet you found online.
