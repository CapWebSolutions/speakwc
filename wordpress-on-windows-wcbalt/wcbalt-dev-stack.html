My Development Environment

The primary tool in my dev is Local by flywheel from Flywheel. Flywheel purchased the premium OS X tool from Pressmatic in December 2016 and ported it to multiple environments. I had previously used Desktopserver from ServerPress, and still use it for sites being developed for older PHP environments. I am able to run Local by Flywheel on a Windows Surface Pro 4 tablet running Windows 10 Pro with 8 GB RAM.
